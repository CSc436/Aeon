Property Detail
bottomproperty
bottom:Number

The position, in pixels, of the bottom edge relative to the bottom anchor, or, if there is no bottom anchor, then the position is relative to the bottom edge of the parent container. If this value is NaN, the object's bottom edge will not be anchored.

The default value is NaN.
